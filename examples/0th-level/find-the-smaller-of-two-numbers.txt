Write a function that, given two numbers, returns the smaller of the two numbers.

function:
=========
name: min

parameter
	-	NUM_ONE: number
	- NUM_TWO: number

return: number

pseudo code:
============
if NUM_ONE is smaller than NUM_TWO
	return a
else
	return b

test cases:
===========
min(3, 5) should be 3
min(5, 3) should be 3
min(-5, 3) should be -5
min(-5, -55) should be -55
